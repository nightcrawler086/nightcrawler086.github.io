So, the last few weeks I've been spending some time in the lab.  I deployed a couple nodes wtih KVM to start a learning path that I laid out for myself.  I chose KVM for a few reasons:  to learn KVM (duh), learn more open-source, and no licensing to worry about.  As I've said before, most of my experience is with VMware, so I'll be comparing the two as I go.

### First run

I intially selected Centos 7 for my nodes, but I actually ended up re-deploying them as Ubuntu 14.04.3 LTS.  I had some trouble getting an Ubuntu instance on top of Centos (should have documented that, but I got frustrated).  I haven't had any trouble with my Ubuntu systems yet, so I'm going to stick with them.

My current purposes are rather trivial, but will get more advanced as time goes on.  I'm starting with some more structured learning on Linux in general.  I'm currently following the LPIC-1 track on [Pluralsight](https://pluralsight.com).  I've completed the first course, [Linux Installation and Initial Configuration]().  Most of the course was information I already knew, but I'm trying to solidify my foundation when it comes to Linux.  I'm really focusing on learning two Linux distributions right now:  Centos and Ubuntu.  I've just recently added Debian as well
