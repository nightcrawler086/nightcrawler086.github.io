## a time to make some changes.

This is my first *real* post to this blog, which i kind of set up on a whim.

For the past 5 years, I've been an focusing on infrastructure technology.  I started off with VMware server virtualization, which was my segue into storage.  I began with EMC storage with their VNX/VNXe product line, and in the last year and a half started focusing on Netapp's FAS line.

With the shift to cloud-computing, being a product specialist is well...not what it used to be.  Selling storage arrays seems a bit archaic.  I've been thinking that I should start focusing more up the stack.

With that in mind I've put together a list of things I'd like to learn:
